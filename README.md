# Dayananda Sagar Business School (DSBS) - PGDM Admissions 2026

A premium, high-converting admissions landing page for the **Dayananda Sagar Business School (DSBS)**, Bengaluru. This project features a fully responsive design, custom interactive UI components, and integrated lead-capture widgets to streamline student enrollment for the PGDM 2026 batch.

## 🚀 Key Features

- **Premium Typography & UI**: Designed with Google Fonts (Outfit & Inter) and modern CSS variables for a refined academic aesthetic.
- **Responsive Layout**: Fluid design that renders beautifully on mobile, tablet, and desktop screens.
- **Dynamic CRM Widget Integration**: Fully integrated with the ExtraaEdge CRM widget for real-time lead capture, including a custom fallback handler in case of loading issues.
- **Custom Dropdowns & Consent Control**: Injects and manages localized States & Cities, PGDM Specializations, and explicit DND/NDNC consent checks automatically.
- **Interactive Components**:
  - Sticky navigation bar with scroll offset adjustment.
  - Mobile drawer menu.
  - Interactive FAQ accordion with fluid animations.
  - Floating WhatsApp chat button for direct admissions inquiries.

## 🛠️ Technology Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 with Modern Flexbox and CSS Grid
- **Interactivity**: Vanilla JavaScript (ES6+)
- **Icons**: FontAwesome v6.4.0

## 📦 Getting Started

To view and edit the project locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/Amanrck96/dsbs.git
   ```
2. Open `index.html` directly in your browser, or use a local development server (such as Live Server in VS Code):
   ```bash
   # If you have Node.js and serve installed globally:
   npx serve .
   ```
